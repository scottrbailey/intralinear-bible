# BSB Intralinear Bible — Installation Guide

The BSB Intralinear Bible combines the **Berean Standard Bible** with inline Hebrew and Greek transliteration linked to Strong's concordance. It is available for two Bible apps:

- **MySword** — Android phones and tablets
- **e-Sword** — Android and iOS phones and tablets

---

## MySword (Android)

### Step 1 — Install MySword

1. Open the **Google Play Store** on your Android device.
2. Search for **MySword Bible** and install it.
3. Open MySword and complete any first-run setup.

### Step 2 — Download the Module

1. On your device, open this link to download the module:
   **[BSBi_1-0-0.zip](#)** *(replace with actual release link)*
2. When prompted, save the file to your device's **Downloads** folder.

### Step 3 — Install the Module

1. Open a file manager app on your device (e.g. **Files by Google**).
2. Navigate to your **Downloads** folder and tap the `BSBi_1-0-0.zip` file.
3. Extract (unzip) the file. You should see one or two `.bbl.mybible` files:
   - `BSBi.bbl.mybible` — standard intralinear display
   - `BSBis.bbl.mybible` — stacked display (transliteration above Strong's number)
4. Move or copy the `.bbl.mybible` file(s) you want to use into the MySword module folder:
   - **Internal storage:** `MySword/Bibles/`
   - If the `MySword` folder doesn't exist yet, open MySword first so it can create it.
5. Open (or restart) MySword. The module will appear in the Bible selector at the top of the screen.

### Choosing Between Standard and Stacked

- **Standard (`BSBi`)** — transliteration appears as a superscript after the English word. Tap it to open the Strong's lexicon entry.
- **Stacked (`BSBis`)** — transliteration and Strong's number are stacked vertically above the English word, similar to a traditional interlinear layout.

## e-Sword (Android and iOS)

### Step 1 — Install e-Sword

**Android:**
1. Open the **Google Play Store**.
2. Search for **e-Sword Bible** and install it.

**iOS (iPhone/iPad):**
1. Open the **App Store**.
2. Search for **e-Sword** and install it.

3. Open e-Sword and complete any first-run setup.

### Step 2 — Download the Module

1. On your device, open this link to download the module:
   **[BSBi_1-0-0.zip](#)** *(replace with actual release link)*
2. Save the file to your device's **Downloads** folder (Android) or any accessible location (iOS).

### Step 3 — Install the Module (Android)

1. Open a file manager app on your device.
2. Navigate to your **Downloads** folder and extract the `BSBi_1-0-0.zip` file.
3. You should see a file named `BSBi.bbli`.
4. Move or copy `BSBi.bbli` to the e-Sword module folder:
   - **Internal storage:** `e-Sword/Bibles/`
5. Open (or restart) e-Sword. The module will appear in the Bible selector.

### Step 3 — Install the Module (iOS)

iOS does not allow direct file access the same way Android does. Use one of these methods:

**Method A — Import via Files app:**
1. Download `BSBi_1-0-0.zip` and extract it. You should see `BSBi.bbli`.
2. Open the **Files** app and locate `BSBi.bbli`.
3. Long-press the file and tap **Share**.
4. Scroll through the share options and tap **Copy to e-Sword**.
5. e-Sword will import the module automatically.

**Method B — Import via e-Sword's built-in browser:**
1. Open e-Sword and go to **Downloads** (or the module manager).
2. Use the built-in browser to navigate to the download link and download directly into e-Sword.

## Features

- **Transliteration** — Hebrew and Greek words are shown in Latin script so you can pronounce them without knowing the original alphabets.
- **Strong's Links** — tap any transliteration to open the corresponding Strong's lexicon entry explaining the word's meaning and usage.
- **Translator Notes** — footnotes from the BSB translation team are included and accessible by tapping the note markers.
- **Cross-References** — Treasury of Scripture Knowledge (TSK) cross-references are included (e-Sword only).
- **Section Headers** — optional section headings from the BSB are included and can be toggled in app settings.

## Troubleshooting

**The module doesn't appear after installation.**
Restart the app. If it still doesn't appear, verify the file is in the correct folder and has the correct extension (`.bbl.mybible` for MySword, `.bbli` for e-Sword).

**Tapping a transliteration doesn't open the lexicon.**
Make sure you have a Strong's dictionary installed in the app. MySword and e-Sword both offer free Strong's dictionaries in their download sections.

**The Hebrew or Greek text looks like boxes.**
Your device's default font may not include Hebrew or Greek characters. This affects the script display only — the transliteration and Strong's links will still work normally.
